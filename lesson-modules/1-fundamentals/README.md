# Module 1: Claude Code Fundamentals

**Duration:** ~2 hours | **Prerequisites:** Claude Code installed

This module teaches you the core mechanics of Claude Code using FinTrack, a fictional personal finance SaaS startup. By the end, you'll know how to use Claude Code's most powerful features as a PM.

---

## Modules

### 1.1 — Welcome
`/start-1-1` · 10 min

Your introduction to Claude Code and the FinTrack case study. Claude walks you through the project structure and what's ahead.

### 1.2 — Visualizing Files
`/start-1-2` · 15 min

Set up your visual workspace. Learn how to explore project files and use Claude's file tree effectively.

### 1.3 — First Tasks
`/start-1-3` · 25 min

The core PM skill loop: read files → analyze → produce output. You'll work with user interviews, meeting notes, and communication style files to produce real deliverables.

**Files used:**
- `user-interviews/` — 8 customer interview transcripts
- `meeting-notes-raw.md` — unstructured meeting notes
- `communication-styles/` — executive email, Slack update, Notion doc examples

### 1.4 — Agents
`/start-1-4` · 20 min

Clone Claude to run multiple tasks in parallel. You'll dispatch agents to summarize support tickets, analyze competitor research, and synthesize meeting notes — simultaneously.

**Files used:**
- `support-tickets/` — 10 support tickets
- `competitor-research/` — 5 competitor profiles
- `meeting-notes/` — 10 meeting notes
- `user-interviews/` — 5 customer interviews

### 1.5 — Custom Sub-Agents
`/start-1-5` · 15 min

Create specialized agents with custom personas. You'll work with the built-in Engineer, Executive, and User Researcher agents to evaluate a feature spec from three different perspectives.

**Files used:**
- `feature-spec-realtime-collab.md`

### 1.6 — Project Memory
`/start-1-6` · 15 min

Learn how `CLAUDE.md` files give Claude persistent context across sessions. You'll see how to use project memory to avoid re-explaining your company, goals, and preferences every time.

### 1.7 — Navigation
`/start-1-7` · 15 min

Keyboard shortcuts, navigation tricks, and Claude Code UI features that make you faster.

---

## What You'll Produce

- Synthesized customer insights from 8 user interviews
- A reformatted meeting summary in three different communication styles
- A multi-perspective feature analysis (engineer, executive, user researcher)
- Your own `CLAUDE.md` project memory file
