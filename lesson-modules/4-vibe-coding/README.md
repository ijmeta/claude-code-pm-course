# Module 4: Vibe Coding

**Duration:** ~1.5 hours | **Prerequisites:** GitHub account, Vercel account

Build and ship a real web application from scratch — no prior coding experience required. Claude guides you through every step, from defining requirements to a live URL.

---

## What You'll Build

A **personal budget tracker** web app tailored to your own requirements. By the end of Module 4.5, you'll have a working app deployed at a real public URL.

---

## Modules

### 4.1 — Setup
`/start-4-1` · 10 min

The vibecoding mindset: how to think about building with AI, what to expect, and how to set up your project folder. Claude explains what's about to happen and gets your environment ready.

### 4.2 — Plan
`/start-4-2` · 25 min

Claude interviews you about what you want your app to do. You answer questions, Claude synthesizes your answers into a requirements document, and you align on scope before writing any code.

**Output:** A `requirements.md` file defining your app

### 4.3 — Build & Iterate
`/start-4-3` · 40 min

The main build phase. Claude scaffolds the app based on your requirements, then you iterate together — reviewing screenshots, requesting changes, and refining until it works the way you want.

**What Claude does:**
- Generates the full app code
- Runs the app locally and takes screenshots
- Revises based on your feedback
- Handles bugs as they come up

### 4.4 — GitHub
`/start-4-4` · 15 min

Push your project to GitHub for version control and backup. Claude walks you through initializing git, creating a repo, and making your first commit — even if you've never used git before.

**What you'll do:**
- Create a GitHub repo for your app
- Commit and push your code
- Learn the basic git workflow

### 4.5 — Go Live
`/start-4-5` · 15 min

Deploy your app to Vercel and get a real public URL you can share. Claude walks you through the Vercel setup, deployment, and any final fixes needed.

**Output:** A live web app at a `*.vercel.app` URL

---

## What You Need

- A [GitHub](https://github.com) account (free)
- A [Vercel](https://vercel.com) account (free)
- That's it — Claude handles the rest

---

## What You'll Produce

- A `requirements.md` defining your app
- A complete working web application (HTML/CSS/JS or React)
- A GitHub repo with your source code
- A live deployed URL
