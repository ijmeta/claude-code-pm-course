# Course Modules

This folder contains all 21+ interactive modules of the Claude Code for Product Managers course.

Each module lives in its own folder and is powered by a `CLAUDE.md` teaching script. When you run a slash command in Claude Code (e.g., `/start-1-1`), Claude reads that script and guides you through the lesson interactively — stopping at checkpoints, waiting for your input, and running exercises with you.

---

## Module Structure

| Folder | Module | Slash Command | Duration |

| `4-vibe-coding/` | Build & Deploy a Web App | `/start-4-1` → `/start-4-5` | ~1.5 hrs |

---

## How to use these modules

1. Navigate to the project root in your terminal
2. Run `claude` to launch Claude Code
3. Type the slash command for the module you want to start (e.g., `/start-1-1`)
4. Claude will read the teaching script and guide you from there

Start at Module 1.1 if you're new. Modules build on each other within each track, but tracks (1, 2, 3, 4) are largely independent.
